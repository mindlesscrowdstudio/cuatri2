Hacer un sistema que pida dos números al usuario y calcule la resta entre el mayor y el menor de los números ingresados. Verificar que los datos ingresados sean números y si no lo son pedirlos nuevamente al usuario. Hacer una función para realizar el cálculo. Escribir la operación realizada y el resultado en la página. Cuando se trate de dos números iguales no realizar la operación y escribir un mensaje. Luego de cada operación preguntar al usuario si desea realizar una nueva operación o concluir el programa
